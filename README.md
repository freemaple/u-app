# 运行到浏览器
运行-运行到浏览器

### 真机运行
运行-运行到手机或模拟器
android: 点击运行到Android App基座 （android手机需要开启开发者模式）
ios: 需要再mac系统 点击运行到ios 

###环境配置
h5: manifest.json h5-devServer-proxy-target http://dev-appserver.dressin.com
app: config.js 测试环境env = "development" 

### 自定义基座
发行-原生APP打包 -选择打自定义基座
运行-运行到手机或模拟器 选择自定义基座运行
